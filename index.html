I'm Possible

Ever wanted to create 2D games, but step back because Roblox doesn't have a 2D physics engine? Well, now its possible! Use Nature2D to create versatile and smooth simulations and mechanics for your 2D games with minimum effort!

Extensive

Nature2D is built for you to create possibly anything you can imagine. Its equipped with RigidBodies, Constraints and smooth collisions to help map out your imagination through code! RigidBodies and constraints can potentially be made with almost all UI elements, from Frames to TextBoxes.

Verlet Integration

The library uses methods of Verlet Integration and Convex Hull Collisions! Create almost anything you can imagine. From bouncy boxes to destructible structures, even character movement in no time.
